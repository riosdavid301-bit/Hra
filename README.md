# HRA Feuille de Temps - Time Sheet Management

A Progressive Web App (PWA) for managing time sheets and hours.

## 📱 Installation on iPhone

### Option 1: Add to Home Screen (Recommended)
1. Open **Safari** on your iPhone
2. Navigate to your app's URL (or GitHub Pages)
3. Tap the **Share** button (↑) at the bottom of the screen
4. Scroll down and select **Add to Home Screen**
5. Enter a name for your app (or use the default)
6. Tap **Add** in the top right corner
7. Your app will now appear on your home screen as an icon

### Option 2: Add to Home Screen (Alternative)
1. In Safari, go to the menu (⋯)
2. Select **Add to Home Screen**
3. Customize the name and tap **Add**

## 💻 Installation on Android

1. Open **Chrome** on your Android device
2. Navigate to your app's URL
3. Tap the menu (⋮) at the top right
4. Select **Install app** or **Add to Home Screen**
5. Confirm the installation

## 🖥️ Desktop Usage

1. Visit the app URL in your browser
2. You'll see an install prompt (if supported)
3. Click **Install** to add to your system

## ✨ Features

- **Responsive Design**: Works on mobile, tablet, and desktop
- **Offline Support**: Access your app even without internet (when implemented)
- **Fast Loading**: PWA technology ensures quick load times
- **App-like Experience**: Runs in standalone mode, looking like a native app

## 📋 Requirements

- Modern web browser with PWA support
- HTTPS connection (required for PWA)
- Web manifest file (manifest.json)
- Service worker file (when implemented)

## 🚀 Deployment

To deploy this app:

1. Push your code to GitHub
2. Enable GitHub Pages in repository settings
3. Set the source branch to main
4. Your app will be available at https://riosdavid301-bit.github.io/Hra/
